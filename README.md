## Descripción breve

Este programa es un juego de secuencias numéricas creado con Tkinter. El jugador debe identificar el siguiente número en diferentes tipos de secuencias (aritméticas, geométricas, Fibonacci, alternantes y cuadrados). 
El juego dura 5 minutos, controlados por un temporizador interno. Cada respuesta se valida, se actualiza el puntaje y aparece una nueva secuencia.
Al finalizar el tiempo, se muestra el puntaje total. El código está estructurado en una clase y cuenta con interfaz gráfica e interacción por botones y entradas.
